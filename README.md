Aim Trainer

A browser-based precision aim trainer built with HTML, CSS, and JavaScript. Just open and play.

------------------------------------------------------------------------------------------------------------------------------
Features

3 Difficulty Modes — Easy, Medium, and Hard with unique target sizes, speeds, and point values
Smooth Target Movement — Targets glide between random positions at 60fps using requestAnimationFrame
Live HUD — Real-time Score, Streak, Accuracy, and Countdown during gameplay
2× Streak Bonus — Land 5 hits in a row to trigger a 5-second double-points window
Grade System — S / A / B / C / D / F rating based on accuracy at the end of each round
End-of-Round Stats — Score, Hits, Misses, Accuracy, Best Streak, and 2× Events
Tactical UI — Gamer-style dark theme with glowing elements
Fully Responsive — Works on desktop, tablet, and mobile
-----------------------------------------------------------------------------------------------------------------------------

Getting Started

No install required. Just clone and open.
cd aim-trainer
Then open index.html in any browser — Chrome, Firefox, Edge, or Safari.

Tip: For the best experience, use a desktop browser with a mouse.
------------------------------------------------------------------------------------------------------------------------------

Project Structure
aim-trainer/
├── index.html    - Page structure and layout
├── style.css     - All styling — theme, animations, responsive design
└── script.js     - Game logic — movement, scoring, timers, effects

------------------------------------------------------------------------------------------------------------------------------
How to Play

Open index.html in your browser
Pick a difficulty — Easy, Medium, or Hard
Hit Start and click the glowing target as fast and accurately as you can
Build a streak of 5 hits in a row to activate 2× points for 5 seconds
When time runs out, your accuracy is graded and stats are displayed
